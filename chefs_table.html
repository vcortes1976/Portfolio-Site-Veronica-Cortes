CHEF'S TABLE

CONCEPT

Chef’s Table is designed for foodies, travellers and food professionals.

With so many restaurant to choose from and so many restaurant apps and tools on the market, it’s hard to separate the good from the bad as customer reviews alone are not always reliable.

Chef’s Table is a responsive web app offering expert recommendations from top chefs around the world, from cheap eats to fine dining.

ROLE

Market research and analysis
User research and testing
Product design
Style guide

DURATION

Approximately 8 weeks.

Process

UNDERSTAND

Competitor analysis
UX analysis

I began by looking at competitor apps to understand their objectives, strategies and marketing advantage; and conducted a SWOT analysis. I also conducted a UX analysis, looking at the usability, layout, navigation structure and user psychology of the apps and websites (if available). This allowed me to identify my differentiators so I could focus on elevating the valuable content using images and colour where appropriate; build trust through consistent and cohesive designs; and connect with the user by personalising and providing a social element to the app.

OBSERVE

User interviews

I conducted a number of user interviews to understand if Chef’s Table would be of value to the user and how it could better serve the user if I understood their needs and requirements. A number of patterns and key insights emerged.

“Chef recommendations are valuable but wouldn’t limit to only this. User ratings are also valuable.”

POINT OF VIEW

User personas

I developed user personas from the user interviews to understand user behaviours, goals and frustrations.

IDEATE

User flows
Sketching wireframes

PROTOTYPE & TEST

Mid-fidelity wireframes
User testing
Iterations

I created mid-fidelity wireframes and conducted user tests to assess the learnability for users interacting with the restaurant finder and booking application for the first time on a mobile device. I wanted to establish and observe if new users understood how to complete basic initial functions such as logging in, searching for and navigating to a desired selection, completing/updating and searching for bookings. I collated the feedback to iterate and improve the design and conducted further user tests with my interviewees.
